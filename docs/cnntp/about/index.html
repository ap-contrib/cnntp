About this site

nntp.perl.org was created by Ask Bjørn Hansen.

The site is built with Apache, mod_perl, Combust, colobus and Net::NNTP.

A bit of Perl code is using Net::NNTP to connect to the colobus server which uses the perl.org ezmlm archives as the "news spool".

The code is available from Subversion at https://svn.develooper.com/projects/cnntp/.
